# Subcontracting Management with Stock Valuation


        This bridge module allows to manage subcontracting with valuation.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-mrp_subcontracting_account
```

## Dependencies

This addon depends on:
- mrp_subcontracting
- mrp_account
