# account_qr_code_sepa


        This module adds support for SEPA Credit Transfer QR-code generation.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-account_qr_code_sepa
```

## Dependencies

This addon depends on:
- account
- base_iban
